A plugin that contains several fully functional 8-bit game blocks.

== Description ==

Transform your WordPress site into an interactive gaming experience with this series of 8-bit game blocks that can be added to any page or post.

Features:
* Complete falling blocks gameplay with all standard pieces

Perfect for gaming websites, entertainment blogs, arcade-style pages, or anywhere you want to add interactive content that keeps visitors engaged.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/eight-bit-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the Falling Blocks block to any post or page using the block editor
4. Customize the game appearance using the block settings if desired
